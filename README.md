## Installation

To run my app with npm

```bash
  npm clone https://github.com/LokeshPereiro/gifs-searcher-app.git
  cd gifs-searcher
  npm install
  npm start
```

To run my app with yarn

```bash
  yarn clone https://github.com/LokeshPereiro/gifs-searcher-app.git
  cd gifs-searcher
  yarn
  yarn dev
```
